# Transport Decision Program 🚗🚲

A beginner Python project created while learning Python through FreeCodeCamp.

## Description

This program decides whether a person can travel based on:

- Distance to travel
- Weather condition
- Availability of bike
- Availability of car
- Availability of ride-share app

The program uses `if`, `elif`, and `else` conditions to make a decision.

## Concepts Used

- Variables
- Boolean values
- Conditional statements
- Comparison operators
- Logical operators (`and`, `or`, `not`)
