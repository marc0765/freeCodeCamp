# Character Creator 🎮

A beginner Python project created while learning Python through FreeCodeCamp.

## Description

This program creates a character profile with a name and three basic stats:

- Strength (STR)
- Intelligence (INT)
- Charisma (CHA)

The program validates:

- If the character name is a string
- If the name is not empty
- If the name length is within the limit
- If the name does not contain spaces
- If all stats are integers
- If stats are between 1 and 4
- If the total starting points equal 7

If all inputs are valid, it generates a character sheet with visual stat bars using filled and empty dots.


## Concepts Used

- Functions
- Variables
- Conditional statements
- String formatting
- Data type checking
- `isinstance()`
- Arithmetic operations
- String multiplication
- Input validation
