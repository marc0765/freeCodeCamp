# Discount Calculator 💰

A beginner Python project created while learning Python through FreeCodeCamp.

## Description

This program calculates the final price after applying a discount.

The program checks:

- If the price is a valid number
- If the discount is a valid number
- If the price is greater than 0
- If the discount is between 0 and 100

If the input is valid, it returns the discounted price.

## Concepts Used

- Functions
- Variables
- Conditional statements
- Data type checking
- `isinstance()`
- Arithmetic operations
