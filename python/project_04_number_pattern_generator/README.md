# Number Pattern Generator 🔢

A beginner Python project created while learning Python through FreeCodeCamp.

## Description

This program generates a sequence of numbers from 1 up to a given number.

The program checks:

- If the input is an integer
- If the number is greater than 0

If the input is valid, it returns a number pattern separated by spaces.

## Concepts Used

- Functions
- Variables
- Conditional statements
- For loops
- Lists
- String conversion
- `isinstance()`
- Input validation
- String joining using `.join()`
